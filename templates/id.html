Districts ID

The data for districts are too huge to scrap. Look the state ID above and replace the <state_ID> with yours on the given link below
 https://cdn-api.co-vin.in/api/v2/admin/location/districts/<state_ID>
Example: https://cdn-api.co-vin.in/api/v2/admin/location/districts/31
